Idea:
This app is a diary app. 

User Stories:
As a user, I can write down my thoughts and experices every day.
When I open the app, I see a list of entries. Every entry is one day.
I can add a new entry for today. When I'm writing a entry, I'm in the editing mode.

Design:
Clean, modern, minimalistic

Data format:
Use Markdown as format. The filename is always DATE_title.md
